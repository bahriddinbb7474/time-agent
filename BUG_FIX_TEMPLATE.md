# Bug Fix Template (Constructor → Codex)

Используется для поиска и исправления ошибок в проекте.

---

## Контекст
Описание ошибки или проблемы.

## Где проявляется
Файл / модуль / команда / сценарий.

## Симптомы
- ошибка в логах
- неправильное поведение
- падение программы

## Текущая задача
1. Найти причину ошибки
2. Определить какие файлы связаны
3. Предложить минимальное исправление

Важно:
На этом этапе код НЕ изменять.

---

## Шаг исправления

После подтверждения конструктора:

1. Изменить только необходимые файлы
2. Внести минимальные правки
3. Не менять архитектуру проекта

---

## Формат ответа Codex

1. Причина ошибки
2. Какие файлы связаны
3. Какие изменения внесены
4. Как проверить исправление
5. Возможные побочные эффекты

После выполнения остановиться и ждать следующую команду.

---

Ограничения:
- не изменять другие файлы
- не выполнять дополнительные действия
